# Changelog

All notable changes to this project will be documented in this file. The format is based on Keep a Changelog and this
project adheres to Semantic Versioning.

## [Unreleased]
- No changes since `v1.0.0`.

## [1.0.0] - 2025-11-29
### Added
- Stacked context menus with AI drilldown from responses, draggable/maximizable windows, and backdrop-aware closing.
- Parallel tab loading plus simple conversation view to keep AI and iframe tools responsive.
- Context-aware defaults, enable toggles, and preconfigured AI tools with smart context wrapping.
- Global default model selector, searchable provider dropdown, and per-provider model caching.
- Two-panel settings layout with dedicated tool editing, storage overview/reset, and About/Changelog pages.
- Reader upgrades: interaction zones with help overlay, volume-key navigation on mobile, and streamlined uploads.
- Iframe rendering now shows loading and error states with URL-encoded context parameters.

### Changed
- Refined AI tools list UI with message highlighting and simplified tool rows.
- Updated loading component styling and context menu layout visuals.
- Repository guidelines moved to `AGENTS.md` with refreshed AI assistant rules.

### Fixed
- Resolved stale text selection and navigation-zone conflicts in the reader.
- Centered context menu popup titles and removed unused code.
